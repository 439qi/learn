# Nihongo Meanings Summary

## 肯定句  

```nihongo
[n.] は　[n.] です。  
```

译作: ~是~  

特别地，当宾语为地点时，也可表达 `在某处` 的含义  

```nihongo
[pron.interro.] でも　[v.posi.]  
```

表明全面肯定，即在任何情况下事态都相同  
此处的 `で` 不可省略，相对应地，表全面否定时，也不可添加 `で`  

## 否定句  

```nihongo
[n.] は [n.] ではありません。  
```

译作: ~不是~  

```nihongo
[pron.interro.] も　[v.neg.]  
```

表全面否定，且按助词不同有如下情况  
`[n.]を[v.]` => `[pron.interro.] も [v.neg.]`  
`[n.]に/から/と[v.]` => `[pron.interro.]に/から/と も [v.neg.]`  
`[n.]へ [v.]` => `[pron.interro.](へ) も [v.neg.]`  

## 疑问句  

```nihongo
[1] [v.masu.] か
[2] [clause.plain.] でしょうか
[3] [n.] は　[n.] ですか，[n.]ですか......
```

句尾 `か` 表疑问  
敬体形疑问句不使用 `?`  
简体形疑问句省略 `か`，句尾使用　`?`  

[2] 与 [1] 表达的含义相同，但礼貌程度更高  
对于二类形容词小句和名词小句，省略 `だ`  

对于 [3] 的回答  

- 肯定回答:  
        当提问有多个选项时，回答不用 `はい`  
        はい，[n.] です。|  
        はい，そうです。  
- 否定回答:  
        いいえ，[n.] ではありません。|  
        いいえ，[n.] じゃありません。|  
        いいえ，<ruby>違<rt>ちが</rt></ruby>います  

## 表从属关系、属性  

```nihongo
[n.] の [n.]
```

大致等同于中文的 `的`  

## 存在句  

```nihongo
[place.]に [sb/sth.]が います/あります  
```

译作: ~有~  
> 特别地，当场所的助词为 `で` 时，译作发生  

```nihongo
[sb/sth.]は [place.]に  います/あります 
```

译作: ~在~  

`あります` 用于不具有意志的事物  
`います` 用于具有意志的事物  

## 比较句  

```nihongo
A[n.]は B[n.]より [adj.]です  
B[n.]より A[n.]のほうが [adj.]です
```

译作: `A` 比起 `B` 更~  

```nihongo
A[n.]は B[n.]ほど [adj.i.]くないです  
A[n.]は B[n.]ほど [adj.na.]ではありません
```

译作: `A` 不及 `B` ~  

```nihongo
[n.]の中で [n.]は 一番 [adj.]です
```

译作: 在~中，~是最~的  

`一番` 必须用于明确指出比较对象或范围的情况下  
> 特别地，当范围名词为时间、场所时，使用 `[n.]で`  

```nihongo
- A[n.]と B[n.]と どちらが [adj.]ですか
- A[n.]のほうが [adj.]です
- どちらも [adj.]です
```

译作:  
\- A 和 B 哪个更~  
\- A 更~  
\- 哪个都~  

```nihongo
[adj.]ほうが いいです
e.g.
子供は 元気な ほう がいいです。
```

例句可译为: 孩子还是精神一点好  
用于比较同一个事物的性质时，其中な形容词后要加 `な`  

```nihongo
[v.ta.]ほうが いいです
```

译作: 还是~好  
用于比较两个动作、行为  
**其否定形式为将 `[v.ta.]` 替换为 `[v.nai.]`**  

## 表示情感  

```nihongo
A[n.]は　B[n.]が　[adj.]です
```

表达情感的形容词适用于如上句式，其中 `A` 表示情感的主体，`B` 表示情感的对象

```nihongo
[v.te.] みる  
```

尝试做某事  

## 为做准备而进行的动作  

```nihongo
[v.te.] おく  
```

该句式用于为某件事做准备而有意识进行的动作

```nihongo

帰りの切符を買っておきますか?  
```

## 为特定目的而进行的动作  

```nihongo
[clause.v.basic.] ために，[clause.]  
[n.] のために，[clause.]   
```

`~ために` 表示目的，前后两个小句的主语相同  

## 表明动作相继发生  

```nihongo
[v.te.] [v.]
e.g.
図書館へ 行って 本を 借りて，家へ 帰ります。
[v.te.]から [v.]
e.g.
家へ帰ってから，もう 一度 日本語の勉強をします。
```

表明两个及以上的动作按照时间顺序相继发生  
其中，`てから` 不能在同一个句子中反复使用  

以上句式通常只连接两到三项，过多会使得句子繁杂难以理解  
此时通常使用 `それから` 拆分为多个句子  

```nihongo
図書館へ行って，本を借ります。それから 家へ帰って，勉強をします。
```

表明一个动作在另一个动作之前发生  

```nihongo
[v.basic.]前に，~
```

表明一个动作在另一个动作之后发生  

```nihongo
[v.ta.]後で，~
```

## 表示反复进行同一动作  

```nihongo
[n.] ばかり [v.]
[v.te.] ばかりいます
```

可译为: 总是，一直  
带有贬义  

## 表明动作同时发生  

```nihongo
[v.conj.I.] ながら [v.]  
```

表示**同一主体**同时进行两个动作，其中后面的动作为主要动作  

## 现在进行时  

```nihongo
[1] [v.te.] いる
[2] [vt.te.] ある
```

动作属性可分为如下  

|动词|说明|
|-|-|
|状态动词|动词本身已是状态，不使用 `~ている/ある`|
|持续动词|动作可持续|
|瞬时动词|动作发生的时间很短暂|
|形动词|不对应时间，必须使用 `~ている/ある`|

> 部分动词既可作持续动词，亦可作瞬时动词  
> e.g. <ruby>着<rt>き</rt>る</ruby>  

对于持续动词，该句式表示动作正在进行，即正在进行时  
对于瞬时动词，该句式表示动作的结果状态  
对于形动词，该句式表示性质，用法更接近于形容词  

该句式还可用于表示持续性的状态以及反复、习惯性的动作  
在表示习惯性的动作时且句中没有表示时间起点的成分，也可使用ます形  

`~てある` 更强调人为的结果，但句型中不涉及动作的主体  

```nihongo
壁に 私が 絵を 掛かって います。
壁に      絵が 掛けて   あります。
```

特别地，对于 `知っていますか` 的回答，必须使用 `知りません`  

在简体形中，`いる` 的 `い` 常常省略  

## 表示产生令人不愉快的结果的动作  

```nihongo
[1] [v.te.] しまいます
[2] [v.conj.II.] ちゃった
e.g.
大切な書類うっかり忘れてしまいました。
```  

有时也仅仅表示动作的完成而没有令人不愉快的意思  
[2] 为 [1] 的口语形式，语气更弱

## 表变化  

```nihongo
[adj.i.conj.] なります
[adj.na./n.] に なります
```

可译为: 变得~  
表示性质或状态的变化  
其中い形容词去 `い` 加 `く`，な形容词和名词加助词 `に`  

```nihongo
[adj.i.conj.] します
[adj.na./n.] に します
```

可译为: 把~变得~  
表示因动作而引起事物性质的变化  
> 特别地，`[n.]にします`可用于餐馆点餐，收银台结账  

## 请求  

```nihongo
[v.te.] ください
[v.te.] くださいませんか
```

表明请求做某事  
其中在后面加 `ませんか` 显得更为礼貌  

## 命令  

```nihongo
[v.impera.]
[v.conj.I.] なさい
[v.te.]
```

其中，[v.te.]最常用  

在紧急情况下，也可下对上用命令形，如 `逃げろ`  
直接引用含有委托、要求等祈使意义的话语或用于解释词语时使用命令形  

女性一般用动词て形  

要求对方救助、帮助自己时，即使是紧急情况也使用动词て形  

`[v.conj.I.] なさい` 相较于命令形稍微客气一些，常用于老师对学生，父母对孩子

## 表示假定的顺接  

```nihongo
[1] [clause.plain.v.basic./clause.plain.v.nai.] と [clause.]
[2] [clause.plain.past.conj.II.]たら，[clause.]  
[3] [clause.polite.past.conj.I.]ましたら/でしたら，[clause.]  
[4] [clause.plain.]なら，[clause.]
[5] [clause.plain.]のなら，[clause.]
[6] [clause.cond.]ば[clause.]
```

表示若前句成立，则有后句的结果  
始める
[1] 常用于表述真理、恒常性状态、反复性状态等，其后不能使用意志或请求的表达方式  
例句如下  

```nihongo
食べ物を食べないと，人間は生きることができません。  
このボタンを押すと，電源が入ります。  
私は，その店へ行くと，いつもコーヒーを飲みます。  
この道をまっすぐ行くと，デパートがあります。  
```

[2] 对于二类形容词和名词的简体过去否定形式的后缀 `~ではなかった` 需要去掉 `は` 再变形  
> `たら` 带有假设前句动作已经发生的意思  
>> コンピュータを使ったら、電源を切っておいてください。  
>> 电脑使用**结束**后，请切断电源。
>>
> `なら` 可以接续各种时态，包括 `~たから`  
>
[3] 是 [2] 的更为礼貌的说法  

[4] 多用于一对方的某种**意志或愿望**为前提提出建议  
[5] 更有说话者收到资讯后，以其作为前提的语感  
但多数情况下可以省略 `の` 为 `なら`  

[6] 若前后小句共用主语且小句1的谓语不是状态性谓语时，句尾不能使用表示意志、希望、命令、请求的表达方式  

## 表示假定的逆接  

```nihongo

```

可译为:即使，无论  
表示当小句1成立时，小句2理应成立但事实上没有成立  
该句式既可用于假定情况，也可用于表示已经发生的情况


对于记得不太确切的事情向对方进行确认  

## 表推测或不确定  

```nihongo
[clause.plain.] かな/かしら/でしょう/だろう/かもしれません。
```

其中，当简体小句的谓语为な形容词或名词的现在肯定形时，省略句尾的 `だ`  

`かな` 原用于自言自语的场合，有听话人在场的场合表示让对方听到自己的自言自语而向对方提供不太确实的信息  
`かしら` 与 `かな` 语义相近，多为女性使用  

`だろう` 为 `でしょう` 的简体形  

`かもしれません` 表示的概率比 `でしょう`/`だろう` 低  

```nihongo
[v.conj.I.]/[adj.i.conj.]/[adj.na.] そうです
[v.conj.I.] そうにありません
[adj.i.conj.]/[adj.na.] そうではありません
なさそうです
よさそうです

~そうな [n.]
~そうに [v.]
```

特别地，当前接 `ない`、`いい` 时，两者分别变形为 `なさ`、`よさ` 再接 `そうです`  

`~そうです` 句式也可用于名词与动词前，两种情况下分别变形为 `そうな`、`そうに`  

用于描述说话人对某种事物样态的观感、或根据某种情形推测事态的发展

## 只有  

```nihongo
[n.]だけ
[n.]しか [neg.]
```

接续在 `~だけ` 后的 `が`、`を` 有时省略  
接续在 `しか` 后的助词省略

两者意思基本相同，但 `~だけ` 只用于表示客观的限定，`~しか` 带有一定的主观感情色彩  
以如下句子为例  

```nihongo
[1] 北京には3日だけ滞在することができます。
[2] 北京には3日しか滞在すりことができません。
```

[1] 表示在北京客观上只能停留3天  
[2] 带有说话人主观上觉得只能停留3天太短的语感  

## 授受动词  

### 表示物品所有权的转移  

```nihongo
A[sb.]は B[sb.]に [sth.]を あげる/くれる/もらう
```

あげる: `我方A或他方A` 给予 `他方B`  
くれる: `他方A` 给予 `我方B`  
もらう: A从B那里收到  

此处，我方与他方的概念是两者与说话人的亲疏关系相比较而言，更亲近于说话人一方的为我方，否则为他方  
若亲疏关系相同，则使用 `あげる`  
如以下例句中的 `妹` 指咲子的妹妹  

```nihongo
咲子は 妹に 動物の絵を あげた
```

`もらう` 不区分亲疏关系  

### 表示动作的转移  

```nihongo
A[sb.]は B[sb.]に [v.te.] あげる/くれる/もらう
```

`~てあげる`: `我方A` 主动为 `他方B` 做某事  
`~てくれる`: `他方A` **主动**为 `我方B` 做某事  
`~てもらう`: `我方A` 请求 `他方B` 做某事，或 `我方A` 承受了 `他方B` 动作带来的恩惠  

`~てあげる` 带有恩赐于对方的意味，因而通常不用于下对上  

特别地，当为疑问句时，`~てくれる` 可用于委托关系亲密的人为自己做某事，且往往用否定形式 `~てくれない` 显得更为礼貌  

对于请求收下某物，即 `もらってくれない` 与 `もらってもらわない`，前者用于收下的对象参与到对话中的情况，后者用于收下的对象未参与对话  

## 说明原因  

```nihongo
[1] [clause.adj.na.]/[n.]で [sentence.]
[2] [clause.v.te.]/[clause.adj.i.conj.I.]て [clause.]
[3] [clause.]から，[clause.]
[4] [clause.]ので/んで，[clause.]
```

[1]、[2] 小句2不可为祈使句

[2] 也用于[表示动作相继发生](#表明动作相继发生)  

[3] 可用于敬体小句，也可用于简体小句，当为名词或二类形容词简体小句时，`だ` 换为 `な`  
`ので` 比 `から` 更为礼貌郑重，多用于正式场合  
`んで` 是 `ので` 的口语化  

特别地，当第二个小句为提议时，只能使用 `から`  

## 询问与说明理由  

```nihongo
[1] どうして ですか。
[2] [clause.plain.] のです/んです。
[3] どうして [clause.plain.] のですか/んですか。  
```

其中 [2]、[3] 中简体小句谓语为名词和な形容词的现在肯定形时，句末的 `だ` 换为 `な`  

[2] 句式用于说明状况或理由，`~のです` 用于书面语，`~んです` 用于口语  

对于 [3] 句式的回答要使用 [2] 句式  

## 简体小句作定语修饰名词  

小句的主语助词使用 `が`  

小句谓语为动词时，直接接名词  
小句谓语为い形容词时，直接接名词  
小句谓语为な形容词现在肯定形时，接な+名词，否则直接接名词  
小句谓语为名词时，接の+名词  

特别地，当修饰的名词为 <ruby>`時`<rt>とき</rt></ruby> 时  

- 若小句谓语为动词  
  - 小句动词为过去形时，表明主句动作发生时小句动作已完成  
  - 小句动词为现在形时，表明主句动作发生时小句动作尚未完成  
- 若小句谓语为形容词/名词，小句动作表示主句动作进行的时间  
- 其后接 `に` 即 `時に` 表示一次性的，个别的情况  

## 动词简体小句名词化  

```nihongo
[clause.plain.v.]の/こと
```

**动词**简体小句接 `の`/`こと` 使其名词化  

多数情况下，`の` 与 `こと` 可互换，以下情况除外  

- 只可使用 `の` 的情况  

  - 作为表示感知的动词的宾语  
    如 `見る`、`聞く` 等  
  - 作为直接作用于动作对象的动词的宾语  
    如 `手伝う`、`やめる`、`防ぐ` 等  
  - 作主语  
    `Xのは/が Yだ`
- 只可使用 `こと` 的情况  
  - 作为表示语言行为的动词的宾语  
    如 `知らせる`、`伝える`、`話す` 等
  - 特定组合  
    如 [`ことができる`](#表明许可或禁止) [`ことがある`](#存在句) [`ことにする`](#表示已作出的决定) `ことだ` 等
  - 作 `です` 的宾语  
    `Xは Yことです`

## 解释或下定义  

```nihongo
A[n./clause.] という B[n.]
```

用于对句子的其他成分作解释或下定义  
大致可译为: A 这个 B，A 的 B  

此处B为A的类别  
此处小句可为简体形，命令形，禁止形  

在口语中，`~という人は` 也可以说成 `~って`  
若前面的音节为 `ん` 时，可以省略中间的促音，称为促音脱落  

用例如下：  

```nihongo
[1] 私は東京の世田谷というところに住んでいます。
[2] 田中という名前は日本人の名前です。
[3] この標識はタバコを吸うなという意味です。
```

译为:  
[1] 我住在东京的叫世田谷的地方。  
[2] 田中这个姓是日本人的姓。  
[3] 这个标识是禁止停车的意思。  

## 表明自己的意志  

```nihongo
[1] [v.voli.]
[2] [v.voli.] と思います
[3] [v.voli.] と思っています
[4] [v.basic.]/[v.nai.] つもりです
[5] [v.voli.] とする
```

表示当场形成的意志时只能单独使用意志形，即 [1]  

意志形一般不单独使用，而是使用 [2]、[3] 形式  
[2]、[3] 形式比单纯的意志形更礼貌、温和  

[3] 带有决心或意志尚未形成的语感  

[4] 意思同 [3]  

## 表示已作出的决定  

```nihongo
[1] [v.basic.][v.nai.]/[n.] ことにします/しました
[2] [v.basic.][v.nai.]/[n.] ことになりました
```

此处的 `こと` 不能替换为 `の`  

关于动作性名词，如 `留学`、`勉強`、`掃除` 不能直接使用，必须使用 `~すること` 的形式  

[1] 表示**说话人自己**作出的决定  
[2] 表示因为**外在原因**形成的决定这个**结果**，但也可用于**委婉地**表达自己作出的决定  

> 与意志形相比，意志形侧重于表达个人的意图，而 `ことにします` 则强调已作出的决定
