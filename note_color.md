# 一、色彩

# 1. 概念定义

- 色彩  
  色光作用于人眼所引起的除形象以外的视觉特征。人眼所看到的所有波长的光波构成了连续的可见光谱
- 色相 hue  
  颜色的基本特征
- 饱和度 sataration  
  颜色的纯洁度，掺入白光成分越多，颜色越不饱和
- 亮度/明度 brightness  
  人眼所感受到的明暗程度，表明一个物体反射的光亮的属性
- 色彩空间/色彩模型  
  以强度值来表示颜色的模式，指定以 n 维空间来表示颜色值
- 色域  
  即记录和复制色彩的最大范围

# 2. 色彩空间

依据是否依赖于设备可以分为设备相关色彩空间 color dependent color space 和设备无关色彩空间 color independent color space  
不同设备描述颜色所使用的数字量不同，即设备相关空间的坐标各不相同。故客观上相同的颜色，在不同设备上要使用不同的数字量描述  
设备无关空间的坐标是固定的

依据复制色彩方法的不同可以分为加色法空间和减色法空间

- RGB  
  以 R、G、B 三种颜色作为基色进行叠加模拟色彩组合  
  对于位深度为 24 的像素，每一颜色分量值域为 0-255  
  RGB 通过自身发光来呈现色彩，即加色空间
- CMY(K)  
  以靛青**C**yan、品红**M**agenta、黄**Y**ellow 三种颜料作为基色，控制各染料强度以产生新的颜色，由于油墨不纯，需要加上黑 Blac**k**墨产生纯黑色  
  对于像素，每一分量的用油墨各自百分比表示，即 0%-100%  
  CMYK 通过墨点反射光来呈现色彩，油墨越多，被吸收的光就越多，最终颜色就越暗，即减色空间  
  > RGB与CMYK转换  
    C = G+B = W-R  
    M = R+B = W-G  
    Y = R+G = W-B  
- CIE  
  - CIEXYZ  
    XYZ是假想的理想三原色  
    XYZ不为负  
  - CIELab  
    Lab以亮度和色相描述色彩  
    其分量包括L(亮度)，a，b。亮度100为白色，0为黑色。a的分布由绿色到红色，b的分布由蓝色到黄色

# 3. 色彩管理

指通过科学化及数字化的方法，将各种设备校正，并将设备的色彩特征记录于“特征文件”中，从而在设备上得到可预知的色彩，将色彩重现于不同的输出环境下。
