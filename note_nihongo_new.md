## 一、音调  

### 标记方法  

标记方法分为两种，单词上方划线和数字  
划线如 $\underline{\text{あ}}$ $\overline{\text{お}}$ $\underline{\text{い}}$  
数字如 あおい ②  
> 数字说明的是在何处开始出现从高到低的声调，索引从1开始  
>
### 规则  

1. 第一个音和第二个音的声调必定不同  

    > 但当第一个和第二个假名组成长音时，发声时两者均为高音  

2. 出现降音就不会再升回去(0型无降音)  

### 0型和2型  

在三个音以下时，两者发音相同，但其后接其他句子成分时，声调不同  

## 二、文体  
  
使用敬体形的问题为敬体，使用简体形的文体为简体  

敬体形常用于: 正式场合，书信  
简体形常用于: 与亲近的人谈话，有关法律的文章、报纸、商业文件  

## 二、语法  

### 助词  

日语中，紧随名词后的助词用于提示句子成分  

#### <ruby>は<rt>wa</rt></ruby>  

1. 提示主语  
2. 表对比  
    此时 <ruby>`は`<rt>wa</rt></ruby> 读音要重一些  
    可单独使用，亦可接在其他助词后构成复合形式  
    特别地，当助词为 `を` 时，必须去掉 `を`  

    ```nihongo
    韓国へは 行きました。中国へは 行きませんでした。  
    私の部屋には 電話が ありません。
    私は コーヒーは 飲みません。
    ```

3. 表示量词最低限度  
    接续在量词后且与肯定形使用时，表示该数量为最低限度  
    可译为：至少  

#### <ruby>を<rt>o</rt></ruby>  

1. 提示动作(他动词)的对象  
2. 表明经过的场所  

    ```nihongo
    e.g.
    このバスは 駅前を 通る
    ```

    常搭配的动词有 `通る` `渡る` `過ぎる`  
3. 表明离开的场所  

    ```nihongo
    e.g.
    毎朝七時に家を出ます
    ```

    常搭配的动词有 `出る` `卒業する`  

#### も  

1. 也  

    ```nihongo  
    [n.]も　[v.]  
    ```

    多个 `も` 表示"都"  

    ```nihongo
    ファックスも メールも 届きました。
    ```

    可用于[全面否定句](#否定句)  

#### と  

1. 表列举(列举所有对象)  

    ```nihongo
    [n.]と[n.]と 
    ```

    特别地，最后一个 `と` 可省略  
2. 提示共同做某事的对象  

    ```nihongo
    [n.]は　[n.]と　[v.]
    ```

3. 用于简体小句，表示思考或转述的内容  

    ```nihongo
    [clause.plain.] と 思います
    ```

    特别地，`思います` 只可用于表达说话人的思考内容，`思いました`、`思っています`即可表达说话人的思考内容，也可表达其他人的思考内容  

    ```nihongo
    [sb.]は [sb.]に [clause.plain.]と 言いました
    ```  

    对于反复说起的内容，用 `言っています`  

    在口语中，此处的 `と` 也可以说成 `って`  
    若前面的音节为 `ん` 时，可以省略中间的促音，称为促音脱落  

4. 表示互动动词的对象  

    ```nihongo
    田中さんは 佐藤さんと 結婚しました。
    ```

    `結婚する`、`喧嘩する`、`会う` 等  

    其中，`会う` 的对象即可用 [`に`](#に) 也可用 `と`  

#### や  

1. 表列举(列举其中两个对象)  

#### ね  

1. 表征询同意  

    ```nihongo
    [sentence.] ね
    ```

    用于句尾，读升调  
    表征询听话人的同意  
2. 表示同意对方的**提议**  

    ```nihongo
    そうですね。
    ```

#### に  

1. 提示时间点  

    ```nihongo
    [time.]に
    ```

2. 与 <code><ruby>中<rt>じゅう</rt></ruby></code> 共同使用表示期间结束之前  

    ```nihongo
    [time.]中に
    ```

3. 提示存在的场所/主语所在的场所  
4. 提示他动词的间接宾语  

    ```nihongo
    A[sb.]は B[sb.]に [sth.]を　あげる
    A[sb.]は B[sb.]に [sth.]を　もらう 
    ```

    分别译作:  
    A送给B某物  
    A从B得到某物  

    特别地，第二句例句中的 `に` 也可替换为 `から`  
    `から` 常用于组织或团体的情况  
    `もらう` 也用于收到电话、传真、邮件等通信手段  
5. 提示动作的对象  

    ```nihongo
    友達に 会います
    ```

6. 提示人或物体的附着点  
    通常用于人或物体停留在交通工具或椅子上时  

    ```nihongo
    電車に 乗ります　　
    自転車に 乗ります
    ボートに 乗ります
    椅子に 座ってください
    鞄に 書類を入れます
    ```

7. 提示移动行为的目的地，同 [`へ`](#へ)  
8. 表示用途  

    ```nihongo
    [n.]に
    [v.plain.]のに
    ```

    其后一般接 `使う` 等动词  

    ```nihongo
    この写真はパスポートの申請に使いますか。
    > 这个照片是用来申请护照的。
    ```

9. 表明基准  

    ```nihongo
    [n.]に
    [v.basic.]のに
    ```

    前面的名词为基准，后面一般为表示评价的形容词  
    可译为: 对于，关于  

    ```nihongo
    このマンションは買い物に便利です。  
    大人には易しいです。しかし，子供には難しいです。
    ```

#### で  

1. 提示交通方式  
    特别地，步行为 `歩いて`　　
2. 提示动作进行的场所  
3. 提示动作进行的手段及原材料  
    可译为: "用"  

    ```nihongo
    日本語で　手紙を　書きます。
    新聞紙で　紙飛行機を　作りました。
    何で　うどんを　作りますか。
    ```

4. 用于[量词](#量词)后  
5. 表な形容词、名词并列  

    ```nihongo
    僕は 人間で 光です。
    ```

6. 表示动作主体的样态  

    ```nihongo
    みんなで キョーザを 作ります。
    先月 クラス 全員で キャンプへ 行きました。
    ```

7. 表示原因  
    [n.] で

#### へ  

1. 提示移动行为的目的地  

#### が  

1. 提示小主语  
2. 表示小句的转折关系  
    可译作: 但是，可是  

    ```nihongo
    料理は 美味しいですが，ちょっと高いです。
    ```

    口语中可替换为 `けど`  
3. 表示小句的铺垫关系  

    ```nihongo
    李さんを探しているんですが，どこにいますか。
    ```

    口语中可替换为 `けど`  

#### か  

1. 表示敬体疑问句  

    ```nihongo
    [sentence.]か
    ```

2. 表示对若干名词进行选择  
    可译为 "或者"  

    ```nihongo
    [n.]か[n.]
    ```

3. 用于简体小句，表示不确定的内容  

    ```nihongo
    私は この料理が辛いか どうか 知りません。
    私は この料理が辛いか 辛くないか 知りません。
    ```

    可译为: 我不知道这道料理是否很辣  
    小句的主语必须用 `が`  

    当 `か` 前面为な形容词或名词作谓语的小句时，省略 `だ`  
    其后可接 `どうか` 或ない形，含义相同  

    ```nihongo
    どの料理が辛いか 知りません。
    ```

    可译为:我不知道哪道菜辣。  
    用于疑问词作主语的小句  
    小句的主语必须用 `が`  
    当 `か` 前面为な形容词或名词作谓语的小句时，省略 `だ`  

#### から  

1. 提示时间段的起点  
2. 提示移动行为的起点  
3. 表原因，用于陈述原因、理由的句节的句尾  

#### まで  

1. 提示时间段的终点  
2. 提示移动行为的目的地  

#### の  

仅用于简体形  
要求说明或确认某事  

```nihongo
寒いの？
```

### 动词及变形  

#### 自动词与他动词  

日语中的动词可分为自动词和他动词，可类比不及物动词与及物动词  

自动词含有主语自主地进行动作的含义，不带直接宾语  
自动词的重点在于动作的主体  

其中他动词具有主语作用于直接宾语而使其变化的含义，宾语接续助词 `を`  
> [`を`](#をo) 提示经过、离开的场所时，其提示的名词并非宾语  
>
他动词的重点在于执行动作的主体  

**部分**自动词在词形上具有对应关系，以下列举部分  

- 自动词以 `aru` 结尾，他动词以 `eru` 结尾  
- 自动词以 `eru`、`ru` 结尾，他动词以 `su` 结尾  
- 自动词与他动词完全同形  

#### 动词敬体形与简体形  

敬体形即为ます形  
简体形如下所示  

||现在|过去|
|-|-|-|
|肯定|基本形|た形|
|否定|ない形|~なかった|

特别地，`ある` 的现在否定形为 `ない`，过去否定形为 `なかった`  

在动词简体肯定形后加 `って` 表达听说，传闻的意思  

#### 动词原形  

|分类|特点|
|-|-|
|一类动词/五段动词|1.う段非る动词<br>2.あ、う、お段+る<br>3.特例|
|二类动词/上一段动词、下一段动词|上一段动词：い段+る<br>下一段动词：え段+る|
|三类动词/カ变动词、サ变动词|カ变动词：<ruby>来<rt>く</rt>る</ruby><br>サ变动词：する、名词+する|

来る、（名词+）する 必定是三类

倒数第二个假名不是い、え段 必定是一类

### 形容词及变形  

#### い形容词  

以平假名 `い` 结尾的形容词  
> 特例: 嫌い  
い形容词表示静止的状态  

作表语时加 `です`  
作定语时直接修饰名词，不可加 `の`  

变形规则如下  

||现在|过去|
|-|-|-|
|肯定|~いです|~かったです|
|否定|~くないです/<br>~くありません|~くなかったです/<br>~くありませんでした|

> 特别地，`いい` 变形时视作 `よい`  
>
> ||现在|过去|
> |-|-|-|
> |肯定|いいです|よかったです|
> |否定|よくないです/<br>よくありません|よくなかったです/<br>よくありませんでした|
>

#### な形容词  

不以平假名结尾 `い` 的形容词  
> 特例: 嫌い  

作表语时加 `です`  
作定语时与名词间加 `な`  

作表语时变形体现在其后的 `です` 上  

||现在|过去|
|-|-|-|
|肯定|~です|~でした|
|否定|~ではありません|~ではありませんでした|

#### 形容词て形  

い形容词て形: 去 `い` 加 `くて`  
な形容词て形: 加 `で`  

用于两个及以上形容词并列，除最后一个其余形容词为 `て` 形  

#### 形容词简体形  

形容词本身没有简体形，作谓语时其谓语形式具有具有简体形  

对于い形容词  

||现在|过去|
|-|-|-|
|肯定|~い|~かった|
|否定|~くない|~くなかった|

对于な形容词  

||现在|过去|
|-|-|-|
|肯定|~だ|~だった|
|否定|~ではない|~ではなかった|

> 名词作谓语的简体形与な形容词格式相同  
>
#### 形容词的连用形  

形容词都各有两种连用形， 称为第一连用形和第二连用形，变形规则如下  

||第一连用形|
|-|-|
|い形容词|~い => ~く|
|な形容词|~だ => ~に|

其中第一连用形具有副词性，用于修饰动词  

### 形容词取代名词  

```nihongo
[adj.i.]の
[adj.na.]なの
```

作定语时且上下文明确的情况下，可用 `の` 指代修饰的名词  

### 形容词名词化

```nihongo
[clause.adj.i.]の
[clause.adj.na.]なの
```

注意，形容词名词化不等同于取代名词的用法

## 三、表达  

### 人称  

|常用表达|人称|说明|
|-|-|-|
| <ruby>私<rt>わたし</rt></ruby>|第一人称||
| <ruby>貴方<rt>あなた</rt></ruby>|第二人称||
| あの<ruby>人<rt>ひと</rt></ruby>| 第三人称||
| あの<ruby>方<rt>かた</rt></ruby>|第三人称|あの人的礼貌说法|

对于非第一人称，往往不用人称代词，而是 `姓+さん`、`姓+君`、对方的职业、职务等来指代，否则显得不太礼貌  

- 人称的省略  
    对于上下文可知的对象，往往省略人称代词  

    ```nihongo
    - (貴方は)吉田さんですか。
    - いいえ，(私は)吉田さんじゃありません。(私は)森田です。
    ```

### 物主代词  

|  |说话人与听话人相距一段距离|听话人与说话人处于同一位置|
| ------- | - | - |
| これ |距说话人近|较近|
| それ |距听话人近|较远|
| あれ |距离两人都远|很远|

这三者为名词性物主代词，其形容词性物主代词分别为 `この` `その` `あの`  
指示方位时，分别为 `ここ` `そこ` `あそこ`  

### 疑问代词  

|名词性疑问代词|Comment|
|-|-|
|<ruby>何<rt>なん</rt></ruby>|指代物品 |
|<ruby>誰<rt>だれ</rt></ruby>|指代人  |
|<ruby>何方<rt>どなた</rt></ruby>| `誰` 的礼貌说法|
| どちら|指代二者之一|
| どれ|三者以上中指代某一个物品，其形容词性物主代词为 `どの`|
| どこ| 指代方位|
| いつ| 三者以上中指代时间点，其后不加 `に`  |

名词性疑问代词+ `か` 表某个  

```nihongo
誰か  某人
いつか  某时
どこか  某地
```

其后助词为 `を` `が` 时省略助词  

|形容词性疑问代词||
|-|-|
| どんな(+[n.])|指代性质|
| 何の  |指代内容和材料|
| どの  |三者以上中指代物品|

### 方位  

```nihongo
[n.] の 方位指示词
```

其中，`の` 不可省略  

常用的方位指示词  
上, 下, 左, 右, 前, <ruby>後<rt>うし</rt></ruby>ろ, 隣, 中, 外  

其中，`上` 特指竖直上方，不具有中文中 "表面" 的含义  

### 数字  

|0-9|优先读法|音读|训读|
|-|-|-|-|

对于小数，读法同中文，即 `X点XXXX`，其中 `.` 发音 `てん`

### 时间  

句中的时间使用助词 [`に`](#に)  
对于 `今` `昨日` `今日` `午前` 等名词，其后不加 `に`  
对于 `~曜日`，可加可不加  
对于疑问代词 `いつ`，其后不加 `に`；对于 `何~` 形式的疑问代词，需要加 `に`  

`ごろ` 接在时间点的词语之后，表示大致的时间范围  
且用于句中时，其后不加助词 `に`  

表示时间的名词组合时，两者通过 `の` 连接，如 `今日の八時`  
特别地，若前面的名词为 `毎X` 时，`の` 可省略，如 `毎日八時`  

对于每小时的30分，可以表述为 `三十分` 或 `半`，如 `八時三十分`/`八時半`  

### 价格  

日文的价格打折使用降价的几成来表示  
如七折在日文中的对应表达为 <ruby>三<rt>さん</rt>割<rt>わり</rt>引<rt>びき</rt></ruby> 或 <ruby>30<rt>さんじゅう</rt>%<rt>パーセント</rt>引<rt>びき</rt></ruby>

## 四、句式  

### 肯定句  

```nihongo
[n.] は　[n.] です。  
```

译作: ~是~  

特别地，当宾语为地点时，也可表达 `在某处` 的含义  

```nihongo
[pron.interro.] でも　[v.posi.]  
```

表明全面肯定，即在任何情况下事态都相同  
此处的 `で` 不可省略，相对应地，表全面否定时，也不可添加 `で`  

### 否定句  

```nihongo
[n.] は [n.] ではありません。  
```

译作: ~不是~  

```nihongo
[pron.interro.] も　[v.neg.]  
```

表全面否定，且按助词不同有如下情况  
`[n.]を[v.]` => `[pron.interro.] も [v.neg.]`  
`[n.]に/から/と[v.]` => `[pron.interro.]に/から/と も [v.neg.]`  
`[n.]へ [v.]` => `[pron.interro.](へ) も [v.neg.]`  

### 疑问句  

```nihongo
[1] [v.masu.] か
[2] [clause.plain.] でしょうか
[3] [n.] は　[n.] ですか，[n.]ですか......
```

句尾 `か` 表疑问  
敬体形疑问句不使用 `?`  
简体形疑问句省略 `か`，句尾使用　`?`  

[2] 与 [1] 表达的含义相同，但礼貌程度更高  
对于二类形容词小句和名词小句，省略 `だ`  

对于 [3] 的回答  

- 肯定回答:  
        当提问有多个选项时，回答不用 `はい`  
        はい，[n.] です。|  
        はい，そうです。  
- 否定回答:  
        いいえ，[n.] ではありません。|  
        いいえ，[n.] じゃありません。|  
        いいえ，<ruby>違<rt>ちが</rt></ruby>います  

### 表从属关系、属性  

```nihongo
[n.] の [n.]
```

大致等同于中文的 `的`  

### 存在句  

```nihongo
[place.]に [sb/sth.]が います/あります  
```

译作: ~有~  
> 特别地，当场所的助词为 `で` 时，译作发生  

```nihongo
[sb/sth.]は [place.]に  います/あります 
```

译作: ~在~  

`あります` 用于不具有意志的事物  
`います` 用于具有意志的事物  

### 比较句  

```nihongo
A[n.]は B[n.]より [adj.]です  
B[n.]より A[n.]のほうが [adj.]です
```

译作: `A` 比起 `B` 更~  

```nihongo
A[n.]は B[n.]ほど [adj.i.]くないです  
A[n.]は B[n.]ほど [adj.na.]ではありません
```

译作: `A` 不及 `B` ~  

```nihongo
[n.]の中で [n.]は 一番 [adj.]です
```

译作: 在~中，~是最~的  

`一番` 必须用于明确指出比较对象或范围的情况下  
> 特别地，当范围名词为时间、场所时，使用 `[n.]で`  

```nihongo
- A[n.]と B[n.]と どちらが [adj.]ですか
- A[n.]のほうが [adj.]です
- どちらも [adj.]です
```

译作:  
\- A 和 B 哪个更~  
\- A 更~  
\- 哪个都~  

```nihongo
[adj.]ほうが いいです
e.g.
子供は 元気な ほう がいいです。
```

例句可译为: 孩子还是精神一点好  
用于比较同一个事物的性质时，其中な形容词后要加 `な`  

```nihongo
[v.ta.]ほうが いいです
```

译作: 还是~好  
用于比较两个动作、行为  
**其否定形式为将 `[v.ta.]` 替换为 `[v.nai.]`**  

### 表示情感  

```nihongo
A[n.]は　B[n.]が　[adj.]です
```

表达情感的形容词适用于如上句式，其中 `A` 表示情感的主体，`B` 表示情感的对象  

### 去做某事  

```nihongo
[place.]へ [v.conj.I.]に 行く/来る
```

表明移动行为的目的  

### 尝试做某事  

```nihongo
[v.te.] みる  
```

尝试做某事  

### 为做准备而进行的动作  

```nihongo
[v.te.] おく  
```

该句式用于为某件事做准备而有意识进行的动作  
有为了目的而将结果留存下来的意思  

```nihongo
お客さんが来る前に，部屋を掃除しておきます。  
帰りの切符を買っておきますか?  
```

### 为特定目的而进行的动作  

```nihongo
[clause.v.basic.] ために，[clause.]  
[n.] のために，[clause.]   
```

`~ために` 表示目的，前后两个小句的主语相同  

### 表明动作相继发生  

```nihongo
[v.te.] [v.]
e.g.
図書館へ 行って 本を 借りて，家へ 帰ります。
[v.te.]から [v.]
e.g.
家へ帰ってから，もう 一度 日本語の勉強をします。
```

表明两个及以上的动作按照时间顺序相继发生  
其中，`てから` 不能在同一个句子中反复使用  

以上句式通常只连接两到三项，过多会使得句子繁杂难以理解  
此时通常使用 `それから` 拆分为多个句子  

```nihongo
図書館へ行って，本を借ります。それから 家へ帰って，勉強をします。
```

表明一个动作在另一个动作之前发生  

```nihongo
[v.basic.]前に，~
```

表明一个动作在另一个动作之后发生  

```nihongo
[v.ta.]後で，~
```

### 表明有因果关系的动作相继发生  

```nihongo
[clause.plain.v.basic./clause.plain.v.nai.] と [clause.]
```

其中前个小句为后者的充分非必要条件  
常用于表述真理、恒常性状态、反复性状态等  
例句如下:  

```nihongo
食べ物を食べないと，人間は生きることができません。
このボタンを押すと，電源が入ります。
私は，その店へ行くと，いつもコーヒーを飲みます。
この道をまっすぐ行くと，デパートがあります。
```

### 表示反复进行同一动作  

```nihongo
[n.] ばかり [v.]
[v.te.] ばかりいます
```

可译为: 总是，一直  
带有贬义  

### 表明动作同时发生  

```nihongo
[v.conj.I.] ながら [v.]  
```

表示**同一主体**同时进行两个动作，其中后面的动作为主要动作  

### 现在进行时  

```nihongo
[1] [v.te.] いる
[2] [vt.te.] ある
```

动作属性可分为如下  

|动词|说明|
|-|-|
|状态动词|动词本身已是状态，不使用 `~ている/ある`|
|持续动词|动作可持续|
|瞬时动词|动作发生的时间很短暂|
|形动词|不对应时间，必须使用 `~ている/ある`|

> 部分动词既可作持续动词，亦可作瞬时动词  
> e.g. <ruby>着<rt>き</rt>る</ruby>  

对于持续动词，该句式表示动作正在进行，即正在进行时  
对于瞬时动词，该句式表示动作的结果状态  
对于形动词，该句式表示性质，用法更接近于形容词  

该句式还可用于表示持续性的状态以及反复、习惯性的动作  
在表示习惯性的动作时且句中没有表示时间起点的成分，也可使用ます形  

`~てある` 更强调人为的结果，但句型中不涉及动作的主体  

```nihongo
壁に 私が 絵を 掛かって います。
壁に      絵が 掛けて   あります。
```

特别地，对于 `知っていますか` 的回答，必须使用 `知りません`  

在简体形中，`いる` 的 `い` 常常省略  

### 表示产生令人不愉快的结果的动作  

```nihongo

e.g.

```  

有时也仅仅表示动作的完成而没有令人不愉快的意思


可译为: 变得~

其中い形容词去 `い` 加 `く`，な形容词和名词加助词 `に`  

```nihongo

```

可译为: 把~变得~

> 特别地，`[n.]にします`可用于餐馆点餐，收银台结账

### 表推测或不确定  

```nihongo

```

其中，当简体小句的谓语为な形容词或名词的现在肯定形时，省略句尾的 `だ`  

`かな` 原用于自言自语的场合，有听话人在场的场合表示让对方听到自己的自言自语而向对方提供不太确实的信息  
`かしら` 与 `かな` 语义相近，多为女性使用  

`だろう` 为 `でしょう` 的简体形  

`かもしれません` 表示的概率比 `でしょう`/`だろう` 低  


接续在 `しか` 后的助词省略

两者意思基本相同，但 `~だけ` 只用于表示客观的限定，`~しか` 带有一定的主观感情色彩  
以如下句子为例  

```nihongo
[1] 北京には3日だけ滞在することができます。
[2] 北京には3日しか滞在すりことができません。
```

[1] 表示在北京客观上只能停留3天  
[2] 带有说话人主观上觉得只能停留3天太短的语感  

### 授受动词  

#### 表示物品所有权的转移  

```nihongo
A[sb.]は B[sb.]に [sth.]を あげる/くれる/もらう
```

あげる: `我方A或他方A` 给予 `他方B`  
くれる: `他方A` 给予 `我方B`  
もらう: A从B那里收到  

此处，我方与他方的概念是两者与说话人的亲疏关系相比较而言，更亲近于说话人一方的为我方，否则为他方  
若亲疏关系相同，则使用 `あげる`  
如以下例句中的 `妹` 指咲子的妹妹  

```nihongo
咲子は 妹に 動物の絵を あげた
```

`もらう` 不区分亲疏关系  

#### 表示动作的转移  

```nihongo
A[sb.]は B[sb.]に [v.te.] あげる/くれる/もらう
```

`~てあげる`: `我方A` 主动为 `他方B` 做某事  
`~てくれる`: `他方A` **主动**为 `我方B` 做某事  
`~てもらう`: `我方A` 请求 `他方B` 做某事，或 `我方A` 承受了 `他方B` 动作带来的恩惠  

`~てあげる` 带有恩赐于对方的意味，因而通常不用于下对上  

特别地，当为疑问句时，`~てくれる` 可用于委托关系亲密的人为自己做某事，且往往用否定形式 `~てくれない` 显得更为礼貌  

对于请求收下某物，即 `もらってくれない` 与 `もらってもらわない`，前者用于收下的对象参与到对话中的情况，后者用于收下的对象未参与对话  

### 说明原因  

```nihongo
[1] [clause.adj.na.]/[n.]で [sentence.]
[2] [clause.v.te.]/[clause.adj.i.conj.I.]て [clause.]
[3] [clause.]から，[clause.]
[4] [clause.]ので/んで，[clause.]
```

[1]、[2] 小句2不可为祈使句

[2] 也用于[表示动作相继发生](#表明动作相继发生)  

[3] 可用于敬体小句，也可用于简体小句，当为名词或二类形容词简体小句时，`だ` 换为 `な`  
`ので` 比 `から` 更为礼貌郑重，多用于正式场合  
`んで` 是 `ので` 的口语化  

特别地，当第二个小句为提议时，只能使用 `から`  

### 询问与说明理由  

```nihongo
[1] どうして ですか。
[2] [clause.plain.] のです/んです。
[3] どうして [clause.plain.] のですか/んですか。  
```

其中 [2]、[3] 中简体小句谓语为名词和な形容词的现在肯定形时，句末的 `だ` 换为 `な`  

[2] 句式用于说明状况或理由，`~のです` 用于书面语，`~んです` 用于口语  

对于 [3] 句式的回答要使用 [2] 句式  

### 简体小句作定语修饰名词  

小句的主语助词使用 `が`  

小句谓语为动词时，直接接名词  
小句谓语为い形容词时，直接接名词  
小句谓语为な形容词现在肯定形时，接な+名词，否则直接接名词  
小句谓语为名词时，接の+名词  

特别地，当修饰的名词为 <ruby>`時`<rt>とき</rt></ruby> 时  

- 若小句谓语为动词  
  - 小句动词为过去形时，表明主句动作发生时小句动作已完成  
  - 小句动词为现在形时，表明主句动作发生时小句动作尚未完成  
- 若小句谓语为形容词/名词，小句动作表示主句动作进行的时间  
- 其后接 `に` 即 `時に` 表示一次性的，个别的情况  

### 动词简体小句名词化  

```nihongo
[clause.plain.v.]の/こと
```

**动词**简体小句接 `の`/`こと` 使其名词化  

多数情况下，`の` 与 `こと` 可互换，以下情况除外  

- 只可使用 `の` 的情况  

  - 作为表示感知的动词的宾语  
    如 `見る`、`聞く` 等  
  - 作为直接作用于动作对象的动词的宾语  
    如 `手伝う`、`やめる`、`防ぐ` 等  
  - 作主语  
    `Xのは/が Yだ`
- 只可使用 `こと` 的情况  
  - 作为表示语言行为的动词的宾语  
    如 `知らせる`、`伝える`、`話す` 等
  - 特定组合  
    如 [`ことができる`](#表明许可或禁止) [`ことがある`](#存在句) [`ことにする`](#表示已作出的决定) `ことだ` 等
  - 作 `です` 的宾语  
    `Xは Yことです`

### 解释或下定义  

```nihongo
A[n./clause.] という B[n.]
```

用于对句子的其他成分作解释或下定义  
大致可译为: A 这个 B，A 的 B  

此处B为A的类别  
此处小句可为简体形，命令形，禁止形  

在口语中，`~という人は` 也可以说成 `~って`  
若前面的音节为 `ん` 时，可以省略中间的促音，称为促音脱落  

用例如下：  

```nihongo
[1] 私は東京の世田谷というところに住んでいます。
[2] 田中という名前は日本人の名前です。
[3] この標識はタバコを吸うなという意味です。
```

译为:  
[1] 我住在东京的叫世田谷的地方。  
[2] 田中这个姓是日本人的姓。  
[3] 这个标识是禁止停车的意思。  

### 表明自己的意志  

```nihongo
[1] [v.voli.]
[2] [v.voli.] と思います
[3] [v.voli.] と思っています
[4] [v.basic.]/[v.nai.] つもりです
[5] [v.voli.] とする
```

表示当场形成的意志时只能单独使用意志形，即 [1]  

意志形一般不单独使用，而是使用 [2]、[3] 形式  
[2]、[3] 形式比单纯的意志形更礼貌、温和  

[3] 带有决心或意志尚未形成的语感  

[4] 意思同 [3]  

### 表示已作出的决定  

```nihongo
[1] [v.basic.][v.nai.]/[n.] ことにします/しました
[2] [v.basic.][v.nai.]/[n.] ことになりました
```

此处的 `こと` 不能替换为 `の`  

关于动作性名词，如 `留学`、`勉強`、`掃除` 不能直接使用，必须使用 `~すること` 的形式  

[1] 表示**说话人自己**作出的决定  
[2] 表示因为**外在原因**形成的决定这个**结果**，但也可用于**委婉地**表达自己作出的决定  

> 与意志形相比，意志形侧重于表达个人的意图，而 `ことにします` 则强调已作出的决定  
>
